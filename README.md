# Duckorv Stringification / 鸭鸭弦化

**Stringification** turns your character into a 2D "paper" version, allowing for unique movement mechanics like gliding and enhanced jumping.

**弦化**将你的角色变成二维的“纸片”形态，并提供独特的移动机制，如滑翔和跳跃。

## Features / 功能

*   **Stringification Mode / 纸片模式**:
    *   Flattens your character model to look like a piece of paper.
    *   将你的角色模型压扁，看起来像一张纸。
    *   Toggle with the configured key (Default: `Ctrl`).
    *   使用配置的按键切换（默认：`Ctrl`）。

*   **Flight Mechanics / 飞行机制**:
    *   **Glide / 滑翔**: Jump at a certain height and press the Stringification key (Default: `Ctrl`) to glide.
    *   **滑翔**：跳跃达到一定高度并点击弦化键(默认Ctrl)即可滑翔。
    *   **Paper Plane / 纸飞机**: Your character tilts forward like a paper plane while gliding.
    *   **纸飞机**：滑翔时，你的角色会像纸飞机一样向前倾斜。
    *   **Obstacle Detection / 障碍物检测**: Hitting a wall will stop the flight and revert you to normal form.
    *   **障碍物检测**：撞到墙壁会停止飞行并恢复正常形态。

*   **Jump / 跳跃**:
    *   Add jump physics for the Game.
    *   为游戏增加跳跃的物理效果。

## Configuration / 配置

You can adjust the following settings in the Mod Settings menu:
你可以在模组设置菜单中调整以下设置：

*   **Toggle Key / 切换按键**: Key to enable/disable Stringification (Default: F).
    *   启用/禁用弦化的按键（默认：F）。
*   **Jump Height / 跳跃高度**: How high you can jump in Paper Mode.
    *   纸片模式下的跳跃高度。
*   **Flight Speed / 飞行速度**: Forward speed while gliding.
    *   滑翔时的前进速度。
*   **Descent Rate / 下降速率**: How fast you fall while gliding.
    *   滑翔时的下落速度。
*   **Flight Pitch / 飞行俯仰角**: The visual tilt angle of the character while gliding.
    *   滑翔时角色的视觉倾斜角度。

## Notes / 注意事项

*   The mod is designed to work with the standard character controller.
    *   本模组专为标准角色控制器设计。
*   Flight is automatically cancelled if you land or hit an obstacle.
    *   如果着陆或撞到障碍物，飞行会自动取消。
